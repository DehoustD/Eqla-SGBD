Toute commande dans mysql doit se terminer par un point virgule ;
Si vous le faites pas mysql ne sait pas que vous avez terminé la commande en cours.

Partie 1 - Copie des fichiers databaseBlindCode.sql   Localite.sql    DatabaseVente.sql
========
1. Ouvrez un terminal.
2. Allez dans le dépôt Eqla-SGBD
3. Copiez les trois fichiers suivants: databaseBlindCode.sql, Localite.sql, DatabaseVente.sql dans un répertoire d'exercices SQL de votre choix.

Partie 2 - Importation des fichiers de base de données
========
1. Allez dans le répertoire d'exercices SQL
2. Connectez-vous au SGBD MySQL: mysql -u root -p
3. Entrez votre mot de passe
4. Importez le script qui va générer la base de données BlindCode: source databaseBlindCode.sql
5. Importez le script qui va générer la base de données Localites: source databaseLocalites.sql
6. Importez le script qui va générer la base de données Ventes (Ca va prendre plus temps: attendez): source databaseVentes.sql
7. Faites un SHOW DATABASES;
8: Vous devriez voir dans la liste des bases de données: BlindCode, Localites, Ventes
9. Si vous avez des erreurs appelez-moi.
